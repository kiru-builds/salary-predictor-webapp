# 💰 AI Salary Predictor Web App

*By Kirthika | AI & Data Science Student*

## 📌 About
A beautiful Machine Learning web application that predicts 
salary based on years of experience, education level, job 
role and country. Features real-time predictions with 
career growth charts!

## 🌍 Live Demo
[Click here to try the app!](#) 

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn (Linear Regression)
- Pandas
- Matplotlib
- NumPy

## 📊 Features
- Beautiful dark gradient theme
- Salary prediction for 4 countries
- Currency conversion (₹, €, S$, $)
- Monthly/Weekly/Daily breakdown
- Career growth projection chart
- 95% model accuracy

## 🌍 Supported Countries
| Country | Currency |
|---------|----------|
| India 🇮🇳 | INR ₹ |
| Germany 🇩🇪 | EUR € |
| Singapore 🇸🇬 | SGD S$ |
| USA 🇺🇸 | USD $ |

pip install streamlit scikit-learn pandas numpy matplotlib
